Este é um projeto de Calculadora simples em Java..."

Testes (JUnit 5): "Foram criados testes unitários com JUnit 5 para a classe Calculadora. Os testes cobrem as 4 operações, incluindo cenários com números positivos, negativos, zero e o tratamento da exceção de divisão por zero, garantindo a validação do software."

Fluxo de Versionamento (Git): "O gerenciamento de código foi feito com Git e GitHub. O desenvolvimento foi isolado em uma branch chamada feature/calculadora. Após a implementação e a passagem dos testes, a branch foi mesclada (merged) à branch main através de um Pull Request, simulando um fluxo de desenvolvimento colaborativo.
